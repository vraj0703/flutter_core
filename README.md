# Flutter Core

A core utility package providing shared logic, models, and extensions for the workspace.

## Key Features

- **Dependency Injection**: Helper setup for `get_it` and `injectable`.
- **Utilities**: Common extensions and helper classes.
- **Networking**: Configured HTTP client and error handling.
- **Base Models**: Shared data models and entities.

## Getting Started

### Prerequisites

- Flutter SDK
- `build_runner` for code generation.

### Installation

Add to your `pubspec.yaml`:

```yaml
dependencies:
  flutter_core:
    git:
      url: https://github.com/vraj0703/flutter_core.git
```

## Usage

Import the package to access core utilities:

```dart
import 'package:flutter_core/flutter_core.dart';
```
